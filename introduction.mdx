## Why use it

- **Sub-second submission.** Direct carrier connections, no email-to-SMS gateways.
- **Per-message DLRs.** Every send returns a `messageid` you can use to confirm delivery.
- **Branded sender IDs.** Approved alphanumeric sender IDs (e.g. `XYRA TECH`) appear in place of a phone number.
- **One API, all networks.** Routing across Safaricom, Airtel, and Telkom is handled for you.

## Base URL

```text
https://bulksms.xyratechnology.co.ke
```

All endpoints sit under `/api/services/`. For example, send-SMS lives at:

```text
POST https://bulksms.xyratechnology.co.ke/api/services/sendsms/
```

<Note>
  Trailing slashes are required. `…/sendsms` will 404 — use `…/sendsms/`.
</Note>
